🕋 UmrahBooking - Complete Umrah Reservation System  

## 🌟 Overview  
UmrahBooking is a fully functional Umrah booking system designed to facilitate hassle-free reservations for Umrah packages, hotels, transportation, and flights. Built with modern web technologies, this platform ensures a seamless user experience with secure payment integration.

## 🚀 Features  
- 🏨 **Hotel Bookings** – Search and book hotels for your Umrah trip.  
- 🚗 **Transportation Services** – Reserve transport options for a smooth journey.  
- ✈️ **Flight Reservations** – Book flights with real-time availability.  
- 🎟 **Umrah Packages** – Various customizable packages for pilgrims.  
- 💳 **Payment Gateway Integration** – Secure and reliable transactions.  
- 👤 **User Authentication** – Registration and login system.  
- 🛠 **Admin Panel** – Manage bookings, users, and payments efficiently.  
- 🎨 **Responsive UI** – Built with Tailwind CSS and other CSS libraries for an elegant look.  

## 🏗 Built With  
- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS, Bootstrap 5, AOS, GSAP  
- **Backend:** PHP, MySQL  
- **Additional Tools:** Payment gateway integration, AJAX, jQuery
